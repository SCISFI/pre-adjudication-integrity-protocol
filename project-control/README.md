# Project Control

This folder contains the governing project-control materials for The Pre-Adjudication Integrity Protocol.

The Track Charter governs clinical, legal, ethical, curriculum, AI-feedback, participant-facing, professional-facing, and app-build decisions.

The build tracker controls execution.

## Operating Rule

No feature, curriculum language, AI prompt, participation summary, or dashboard element should be added unless it preserves the approved Charter boundaries:

- no offense-detail collection
- no legal advice
- no therapy replacement
- no forensic evaluation
- no risk scoring
- no court-outcome promises
- no spouse/partner pressure
- clinician or supervised-provider access only for sensitive reflections
- participation summaries document activity only
