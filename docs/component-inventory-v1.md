## Guardrail Components

These components carry ethical responsibility and should not be modified casually:

- `BoundaryReminder`
- `CrisisNotice`
- `CompletionCheckbox`
- `NeedsAttentionBadge`
- `ParticipationSummaryPanel`
- `AIFeedbackCard`

## Component Rules

Do not create components that imply:

- scoring
- risk
- relapse prediction
- legal status
- court benefit
- success/failure
- spouse access
- automatic intervention
